# Frontend básico de StayHub

Interfaz estática inicial para el proyecto StayHub.

## Qué incluye

- Landing page responsive
- Sección de resumen del sistema
- Vista de endpoints principales
- Diseño moderno con tarjetas y animaciones suaves

## Cómo abrirlo

Podés abrir `frontend/index.html` directamente en el navegador o servir la carpeta con un servidor estático simple:

```bash
cd frontend
python -m http.server 5500
```

Después abrí:

```text
http://localhost:5500
```

## Nota

Este frontend es una base visual. Cuando la base de datos esté terminada y el backend exponga los datos reales, se puede conectar la interfaz a la API y reemplazar el contenido estático por información dinámica.
